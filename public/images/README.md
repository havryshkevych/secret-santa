# Open Graph Images

## og-image.png

Щоб Telegram та інші месенджери красиво відображали посилання запрошення до гри, додайте файл `og-image.png` в цю директорію.

**Вимоги:**
- Розмір: 1200x630 пікселів (рекомендовано)
- Формат: PNG або JPG
- Назва файлу: `og-image.png`

**Що буде відображатися:**
- Назва гри
- Опис гри (якщо є)
- Зображення (ваш og-image.png)

**Якщо файл не буде додано:**
Система автоматично використає placeholder з назвою гри на червоному фоні.

**Приклад створення зображення:**
Можете використовувати онлайн редактори:
- Canva (https://www.canva.com)
- Figma (https://www.figma.com)
- Або будь-який графічний редактор

**Тема:**
Святкова новорічна тематика з елементами Secret Santa - подарунки, ялинки, Санта тощо.
